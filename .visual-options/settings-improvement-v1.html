问题说明
  • input 太宽：当前 width: 100% 使 select 占满 label 右侧全部空间，视觉上松散且横向过长。
  • 重新检测按钮太长：当前按钮独占一行，作为一个小操作显得过于"沉重"，与轻量级的检测动作不匹配。
各方案差异
  • 方案 A：select 加 max-width: 320px，保留左侧 label 布局，重新检测改为右侧文字按钮。改动最小，保守安全。
  • 方案 B：label 放在 input 上方，横向更紧凑；网络设置两项并排；环境区域底部增加工具栏，重新检测放在右下。布局更现代。
  • 方案 C：select 固定 width: 400px（非 100%），重新检测按钮精简为"检测"二字并移到 section header 右侧。最简洁。
