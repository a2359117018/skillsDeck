## ADDED Requirements

### Requirement: 模拟安装百分比进度

安装过程中系统 SHALL 显示一个带模拟百分比数字的进度条。百分比从 0% 开始，以非线性速度递增，安装完成时跳到 100%。

#### Scenario: 安装开始时进度从 0 开始

- **WHEN** 用户点击确认安装，安装流程开始
- **THEN** 进度条显示 0%，并开始以模拟速度递增

#### Scenario: 安装过程中进度持续递增

- **WHEN** 安装正在进行中
- **THEN** 进度条百分比持续递增，初期增速较快（约每 200ms +3-5%），后期减速（约每 300-500ms +0-3%），上限不超过 92%

#### Scenario: 安装成功时跳到 100%

- **WHEN** 安装成功完成
- **THEN** 进度条立即跳到 100%，显示成功状态

#### Scenario: 安装失败时保持当前百分比

- **WHEN** 安装失败
- **THEN** 进度条保持在当前百分比，显示失败状态和错误信息

### Requirement: 进度条显示百分比数字

进度条 SHALL 在旁边或内部显示当前模拟百分比数字（如 "67%"），用户可以直观感知进度。

#### Scenario: 百分比数字随进度更新

- **WHEN** 模拟百分比变化
- **THEN** 显示的数字同步更新为当前百分比值

### Requirement: 取消按钮整合在进度区域

取消按钮 SHALL 显示在进度条附近，样式紧凑，用户可在安装过程中随时取消。

#### Scenario: 安装中显示取消按钮

- **WHEN** 安装正在进行
- **THEN** 取消按钮显示在进度条区域内，用户可以点击取消

#### Scenario: 取消安装后进度停止

- **WHEN** 用户点击取消按钮
- **THEN** 进度条停止递增，显示"已取消"状态，安装进程被终止
